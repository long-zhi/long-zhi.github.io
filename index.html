“你的后娘出价两千，买我来杀你。”一把银色的小手枪抵在他脖子的动脉上，刘子贺扭头看向持枪的女人，一动，枪被压得更紧了些，“别紧张。如果你能出价四千，我可以放过你。”

夜已深，在香港去往上海的轮船上，除了零星几对赏夜的情人，其余人已然熟睡。刘子贺眯起眼，看着这高挑的女人走近自己，在旁人偶尔擦身经过时顺势依偎进自己的怀中。海风撩拨起她的几缕散发，而她一脸的风情模样。

十天前，刘太太找到她，以两千大洋买她继子的命，顺带着友情附赠她一张船票和一把手枪。

五天前，她只身上了船。经过日夜跟踪，她终于在今夜寻到他落单的时机。

练溪吸了口他手上拿着的烟，红唇吐雾，靡靡地笼在他俩身边。

刘子贺一手搂住她的腰，一手捂住枪口：“等我回上海继承了遗产，你想要多少还不是一句话的事？”

刘家公子哥儿，近年来不在上海活动，但关于他的传闻可是不少。他早些年流连烟花巷，徘徊斗金场。直到后来被他的父亲赶出上海，那个一掷千金的刘家少爷仍活在软香媚肉的舌尖口中。

是了，刘子贺是大房生的嫡子，他十八岁时，生母去世，侧房被扶正。如今的刘太太，正是刘子贺的后母。

时值刘老爷病逝，刘家遗产的着落成了大问题，按着大家族的规矩，刘子贺当是继承人。于是刘太太买凶杀人，在他回沪的船上，企图让他永葬汪洋。

民国临水照花人

发表于 2020-03-07 分类于 文章
本文字数： 752 阅读时长 ≈ 1 分钟

笙歌归院落, 灯火下楼台。民国就是一场散去的戏,曾经锣鼓喧天的倾城故事,早已湮没在落落风尘中，不知所往。那个被光阴抛掷的女子, 又从远年的巷陌,款款走了出来。

她着一袭素锦旗袍, 穿越民国烟雨,走过季节轮回,成为那潋滟红尘的临水照花人。

世间曾有张爱玲, 世间唯有张爱玲。

张爱玲的一生从兴到衰，走过了人生的最繁华; 经受了人生的最苍凉, 她已洞穿人世沧桑,在时空的轮回中升华。她给后人留下的只是几本小说，但这些小说当时篇篇震动文坛。她的文学创作瞬时登上了文学高楼，红遍上海。

她留给世人的又不仅仅是几本小说，还有她绝代无双的风姿和遗世独立的风骨。

张爱玲的文章中所透露出的丝丝荒凉气息、一缕缕苍凉全是她人生的真实写照。无论是《半生缘》中温柔却软弱、委曲求全，只能用“我们再也回不去了”作为灿烂过后的一声哀鸣的顾曼桢,还是《金锁记》中出场粉亮、余景悲怆的曹七巧，或是《沉香屑:第一炉香》中在一个错误的地方错误的时间爱上了一个错误的人的葛薇龙，抑或是《色·戒》中爱上汉奸易先生，却落下个背叛下场的王佳芝……在我看来，她们都是张爱玲，也都不是张爱玲。张爱玲是世间唯一的临水照花人，犹如“一个美丽而又苍凉的手势”，令人回味无穷。

繁华谢过是苍凉, 1995年,张愛玲谢世于美国,7天后才被人发现,家里空空如洗,身上只着一条薄毯,这般风华绝代的人,竟以这般凄凉的方式凋零。我常常在想,张爱玲弥留之际，可曾想到晩年在病榻上的七巧, 是否也懒得去擦拭眼角的一滴清泪?胡兰成说,张爱玲是民国世界的临水照花人。她无须经历多少世事,这个时代的一切自会与她来交涉,她不漂亮,却能够任何一种姿态倾城，就是这个传奇女子，和月亮结下了一世情缘，生于月圆之时，逝于月圆之时。民国的月亮早已下沉，而她的故事却永远不会结束。

时光无涯，聚散有时，因为懂得，所以慈悲。

此花开尽更无花

发表于 2020-03-07 分类于 文章
本文字数： 1.1k 阅读时长 ≈ 1 分钟

梦回千转，叶落乌啼。于迷蒙中仿佛看到一袭素锦旗袍款款走于烟雨缭绕中的女子。没错，她就是民国的临水照花人，她就是一张爱玲。
人生代代无穷已，江月年年只相似。但恐怕，多了张爱玲这位女子，民国就变了，变得像一袭华美的袍，爬满了蚤子。也许，直到后来，张爱玲那孤傲单薄的身影在我们的脑海中逐渐清晰时，我们才知晓，张爱玲带给我们的不仅是绝美的文字，还有如薄荷般凉薄的记忆。
月色正朦胧。尽管世事变迁，上海给人的纸醉金迷的感觉仿佛永远都不会变。尽管朱楼水榭、碧云烟渚隔绝了我们对往事的怀恋，可每深吸一口清新的空气，仿佛都能品味出民国时上海的璀璨芳华。只是时令徒转，错过的，便只能错过了。
其实张爱玲的往事，一点也不倾城，不过是同激滟红尘中的女子大抵相同罢了，但是张爱玲的往事无比倾城，因为张爱玲自有张爱玲的桃之天天。爱上胡兰成，是情不知所起，一往而深；与桑弧擦肩，是不愿为云淡风轻的相逢低眉垂袖；和赖雅执子之手，是在异国华丽转身的邂逅。不管是心口的的朱砂，还是妖娆的白玫瑰，反正她是真的付出了，是真的锦瑟流年。他们都成了她在战乱的荒芜中，繁华的注脚。
我不知，多年后张爱玲回想起当初胡兰成给的承诺愿“岁月静好，现世安稳”时，清绝的她是否会落泪；我不知，风华绝代的张爱玲知音难觅时是否会备感孤独；我不知，飘零于异乡的张爱玲是否对当时自己的决绝有一丝后悔。我只知，当她得知三毛的死，呢喃“三毛怎么自杀了”时，她是热爱生命的，她愿意观世事冷暖，她情愿大隐于市，静观星辰。
再冷傲的人，怕也是柔情之人。何况是一位女子。我想，张爱玲是高傲的，亦是孤单的。生于簪缨世族，却又偏遭战乱。或许这就是民国，一个浪里沉浮、不可捉模的民国。但只要一想起张爱玲，就会心疼。即使定年华给了她磨炼，可命运却待她太不公平。好在她还有文字，文字的世界虽没有阴睛圆缺，花好月圆，可张爱玲打骨子里热爱它们。她将自己的茕茕光阴，细腻地缝在了她巧心安排的锦绣之中，待我们一一去发现，一一去品味。
有人说，张爱玲所写，都是儿女情长，闺中优俪，虽深情，却不大气。果然还是胡兰成懂她，知道她想要什么。她太需要安稳了，以至于情愿与人世隔绝，不与人来往，不愿涉足外界风雨。所以，张爱玲的文字并没有多深刻，也没有多让人们觉醒。请不要怪她，她只是默默治愈了心灵。因为她是最需要静好却最不静好的那个人。她经历了太多的颠沛流离，尝尽了太多的苦情伤感，以至于不想与任何人有任何纠葛，只想安稳度世。也许，这就是张爱玲，也许，这才是张爱玲。
无论是谁的血，成团扇，涸染了她的江山；无论是多么凉薄的花事，谢了幕，成为过往云烟；无论是多么不解销魂，偿前了债，断了后缘，都不要责备她的青青韶华。因为此花不与群花争，此花开尽更无花。

花开半夏

发表于 2020-03-07 分类于 文章
本文字数： 1.1k 阅读时长 ≈ 1 分钟

一个寒意渐起的中秋，她出生在十里洋场，她是名门之后,血液中流淌的都是贵族气质;又是一个寒意渐起的中秋,她离开去异国他乡，她是迟暮美人，再不愿问人间的香火。

人们都说，她是民国的临水照花人，才情如她，在平仄的文字里,忧伤浅舞、回忆蔓延; 孤独如她,在年华的奔波中,轻触苍凉世事;痴情如她,在绚丽斑斓的上海滩绽放传奇一般的爱情。

张爱玲,一个被玩弄在宿命手掌中的可怜人。 她曾苦苦躲避着流年的回忆,却终躲不过牵绊的丝线。母亲的漂洋过海,父亲的激烈决裂,家族的覆灭衰亡。没落王朝,金银散尽,她只得只身前往香港。战事混乱,地区沦陷,她的成就付之东流。她是失败婚姻的牺牲者,她是没落王朝的苦命儿，她是命运摧残的可怜人。

张爱玲,一个以文字倾国倾城的孤傲女子。 她时而挥洒素雅的散文,时而抒写绝美的爱情。在民国这样一个遍地烟雨的年代,再没有人可以与之媲美。 或许是回忆所牵,她总是喜欢由一个家族、一个故事道尽人间冷暖,她的文字因为现实而更显凄凉,但正是这委婉的语言却有着一种柔软的力量。

张爱玲,一个陷入爱情深潭的痴情女子。他是她素锦年华里最美的相遇。一座城,一笺信,在最深的红尘里遇见，从此,她的生命便有了千千阕歌千千念;她是他寻花问柳中曾经的停留,一份情,一颗心,在最美的时光里邂逅了她，从此他的心底便有了痴情深深深几许……“因为相知，所以懂得。”这是胡兰成写给张爱玲的8个字。“因为懂得，所以慈悲。”这是张爱玲返还胡兰成的一片真情。在动荡的年代，拥有痛苦的童年，断翼的天使急切地需要一个依靠。而胡兰成恰在此时走来,“见了他,她变得很低很低,低到尘埃里,但她心里是欢喜的，从尘埃里开出花来”，或许正是因为相知相惜,她不在乎他是别人嗤之以鼻的汉奸,不在乎他已经拥有妻子儿女,甚至不在乎他在她的生命中只是来来又去去,只是期盼有他的相守与疼爱。凝眸时相思成愁，再回首时依恋成怨。旧时明月，撑不起地久天长的诺言，也留不住海枯石烂的誓言。心上的纠葛，解得开，是结，解不开，是劫。一场轰轰轰烈烈的恋爱终究是要结束的。“你到底是不肯。我想过，我倘使不得不离开你，亦不致寻短见,亦不能够再爱别人，我将只是萎谢了。”她最后用决绝的离开, 挽回了自己的尊严。三年的花事,半途夭折,她的才情与文字随着一生所爱而埋葬。她不再参与喧闹的生活,选择远居国外,隐蔽余年。

别来找我,我亦不去寻你。守着剩下的流年,看一段岁月静好,现世安稳。她抛弃了所有的外物,躲避了所有的目光,在简陋的出租屋中死去。花开半夏,一段美丽与怅然的人生故事落下惟幕。

花开半夏,破土在失败的家庭是命，却无法选择。

花开半夏,含苞在动荡的年代是憾，却无法逃离。

花开半夏,盛放在错误的爱情是悔，却无法重来。

花开半夏,萎谢在安静的他乡是悟，却为时过晚。

那块荒寂的土地，迎来了它新的主人。

天子将其作为封地赠予徐东昂，并且十分体恤他，说道：“朕的诸侯王，怎么可以没有统治的子民呢？辖区内的百姓，都应该搬去那里。”

旨意下达，来的第一个人，是个瘸子，她推着轮椅走了一月才来到的这里。她对徐东昂说：“王啊，我愿意追随您，把您作为我毕生的信仰。”

徐东昂很是感动。

他把瘸子推到了最高的那片废墟上，指着满目荒凉的墟堆，信誓旦旦地说：“这里，未来会是比王城更繁荣的街巷。而你——你这个有远见的人，会见证一个国度的崛起！”

字央知道，他尽在说些胡话。

近日，有道士说，皇上命不过而立。

这本是无从考据的谑言。可魏方行并非足月而生，自小没少经受病痛的磨难，故惜命得很。在苦恼了多日后，他想出了个好法子——若是夜夜不眠，总该匀出许多寿命来吧。

政事上优柔寡断的帝王，唯在多寿之事上这般决然。他让内侍夜里守在自己身边，不让自己安眠，可尚不过第二夜，他便阖了双目呼呼睡去……

殿门被推了开来，他猛地从睡梦中清醒，睁着睡眼看女人施施然来到他身边，明明是从风雨中前来，却不曾有一片衣角被雨水洇湿。

是个厉害的女人啊，魏方行赶紧坐起身来，移开视线，故拿姿态：“哦，是朕的皇后。皇后，你深夜前来是有何要事吗？”

“陛下欲靠不眠来增寿，但似乎并无成效啊。”鹿亭忽视他眼底的怒火，笑意盈盈地接下去说道：“陛下是天子，也是臣妾的夫君，臣妾自当为陛下排忧解难，献上真正能让陛下不眠之法。”

她的话语是如此情真意切，可所有人都知道，这位厉害的皇后心中，有着除陛下以外的另一个男人的存在。但凡存在，必定展露端倪，故心思缜密如她，还是被人捉了把柄……

他们夫妻多年，太过了解彼此，因而他不加掩饰地说道：“若皇后果真有让朕不眠之法，朕自然相信你的忠诚。”

隔日，得帝王承诺的鹿亭伏在金銮殿上，戚戚道：“臣妾对陛下一片忠心，无论后宫亦或前朝都帮陛下打理得井井有条。可偏生有些小人作怪，污蔑臣妾与丞相大人有染。”

少年裘衣鞍马，青春张扬无瑕。

曾经落魄的瞎子摇身一变，单林不承想，五年后的苏平望竟与这十二个光彩夺目的字眼息息相关。

在热闹的街巷，人声嘈杂。她听见许多个声音交织在一起追捧他为“苏画师”，于是她也顾不得什么，朝着马上的他喊道——“苏平望，苏平望啊。”

马上的男人在她身上一定睛，可只逗留了一秒，就转瞬撇开，分明是拿她当陌生人。

单林微一怔忪，街巷已亮起了灯笼，红红的光点连接成并排的两条火龙。她从夜市酒肆的说书人口中得知，如今的苏平望是御用画师，深得皇帝的宠信，就连泰安公主都对他投以青眼。“那位公主心在朝野，她是早已下定了主意要与我联姻，同她的太子哥哥一争天下。”是苏平望的声音。

单林偷偷凑近楼上的雅间，伏在门框子上窃听他与旁人的密谈。谁承想，门突然一开，躲在花瓶后的单林被一双手拽了进去。单林左右打量，里头除了他们二人，哪还有第三个人的存在？她恍然明白，苏平望在自说自话，分明是引她上钩。

此刻，她被苏平望压在木门上，精致的雕花硌得她背疼。近距离端详苏平望清俊的眉眼，单林的心一阵阵猛烈地鼓动，不因什么悸动的女儿情，只为那锃光瓦亮的匕首在她的脖颈生生划出了一道血线。

“你是太子的人？”那张唇一开一合，吐出的六个字让单林云里雾里。

她兀地脑海清明起来，连连表示自己的清白：“我是单林，我是你的人。”

泯生·孰为君心织华年(2)

发表于 2020-03-05 更新于 2020-03-07 分类于 文章
本文字数： 9.1k 阅读时长 ≈ 8 分钟

一

金乌尚未西沉，织里房间的门被突然推开。下人尚未来得及禀报，王横扇已经把她推倒在了软榻上。

他如今已经五十二岁了，气力不比从前，连解开她胸前的衣襟都显得吃力非常。于是，织里主动把心口处的肌肤露出来。她心上有个口子，噗噗往外淌着血，所幸她平时惯着黑衣，因此不必叫人发现了去。

她把王横扇的头按在自己心房，他埋头痴迷地吸吮舔舐。咕噜咕噜吸了织里的血吞咽入肠，他如痴如醉地唤着她：“织里，织里。”

织里跟了他二十七年，知道他今日反常地失控，也知道他失控的缘由。

下人先前禀告说，徐贵妃今晨梳洗之时发现眼旁新添了一道皱纹，是以负责为贵妃养颜的王横扇遭了大祸，被太监剥去裤子打了十板子。五十二岁的男人，被一帮阉人剥了裤子打，这是给他极大的羞辱。

半晌后，埋首的男人停止了吮吸，抬起頭来，目光迷离地望着织里。

此刻，他已经不复五十二岁的衰老之态。下垂的眼尾微微上扬，松弛的皮肤变为紧致，哪怕脸上皱纹如丘壑也骤然被填平如川。

这个模样，让织里不禁以为回到了二十七年之前。那是他二十五岁的时候。彼时他是王城最俊秀的男子，纵然手艺不佳令人尴尬，却因为出众的相貌，开着一家客流盈满的裁缝店。

但二十七年过去，每一年都在他的脸庞划上岁月的痕迹。虽然靠着织里的血维持一时的青春容颜，但不可否认，他已是垂垂老矣。
